Detailed Description

Abstract base class for games in the Arcade project.

Provides a default implementation for entity storage, game state management, and interaction with the rendering engine. Derived game classes can reuse these features and implement specific gameplay logic via the update() method. Key features:

  • Stores entities to be displayed on screen.
  • Handles game-over logic.
  • Manages the score. This class implements the IGame interface and is intended to reduce duplication across all game modules.
