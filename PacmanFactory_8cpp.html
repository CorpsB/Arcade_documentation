Detailed Description

Factory functions to create and destroy the Pacman game instance.

Author
Jason KOENIG
Version
1.0
Date
2025-04-14

This file implements the C interface expected by the Core to dynamically load the Pacman game shared object. It defines:

  • create() to instantiate the game
  • destroy() to delete it
  • getType() to indicate the module type (GAME)
  • getName() to return the name of the game

These functions are used by the Arcade Core loader and allow the Pacman game to be treated as a plugin.
