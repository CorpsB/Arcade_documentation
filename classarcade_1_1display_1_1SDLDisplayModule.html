Detailed Description

SDL-based implementation of the ADisplayModule interface.

This class handles rendering and event processing using the SDL2 library. It is responsible for drawing visual elements such as shapes, sprites, and text, and for capturing input events to be passed back to the core logic. The class uses SDL_Window and SDL_Renderer to manage the window and render context, and also supports TTF font rendering via SDL_ttf.
